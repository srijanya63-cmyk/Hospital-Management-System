The Hospital Management Website is a frontend-based web application designed to streamline the process of browsing doctors, booking appointments, and managing patient interactions in a user-friendly and efficient manner.

 Key Features
 1.Doctor browsing with specialization categories
 2. Appointment cart system (like e-commerce cart)
 3. Booking confirmation workflow
 4. Appointment history dashboard
 5. Data persistence using localStorage
 6. Responsive and user-friendly interface
 7. Dynamic content rendering using JavaScript

Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsive layout
JavaScript (Vanilla JS) – Functionality and logic
localStorage – Client-side data storage
